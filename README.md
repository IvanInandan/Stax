# BeliBites

BeliBits is a recipe sharing collaboration web application built using React, node.js with Express and MongoDB.

## Tech Stack

**Client:** React

**Server:** Node.js, Express.js

**UI:** Tailwind CSS with MantineUI

**FE Libraries:**

- Tanstack Query
- Zustand
- React Router

**BE Libraries:**

- Express
- Nodemon
- bcryptjs
- jsonwebtoken
- Mongoose
- dotenv
- CORS
