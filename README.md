# forked

forked is a full-stack React application to collaborate with other chefs using a version controlled community log.

## Tech Stack

**Client:** React

**Server:** Node.js, Express.js

**UI:** Tailwind CSS with MantineUI

**FE Libraries:**

- Tanstack Query
- Zustand
- React Router

**BE Libraries:**

- Express
- Nodemon
- bcryptjs
- jsonwebtoken
- Mongoose
- dotenv
- CORS
